HTML
MARCAÇÃO
O HTML funciona no que se costuma definir no meio, como uma linguagem client-side (lado do cliente), em contraposição a muitas linguagens que são executadas do lado do servidor, como é o caso do PHP e do ASP.
UNIVC
CSS
ESTILIZAÇÃO
CSS (Cascading Style Sheets) é uma linguagem de estilo que define a aparência de páginas da web. O CSS é usado para estilizar elementos HTML, como a fonte, as cores, os espaçamentos, as margens e a disposição dos objetos na página.

JAVA SCRIPT
PROGRAMAÇÃO
JavaScript (JS) é uma linguagem de programação usada para criar páginas da web interativas. É uma das tecnologias fundamentais da World Wide Web.
UNIVC
CDN
REDE DE ENTREGA
CDN é a sigla para Content Delivery Network, que significa Rede de Entrega de Conteúdo. É uma infraestrutura de servidores que distribuem conteúdo da web de forma mais rápida e próxima dos usuários.
